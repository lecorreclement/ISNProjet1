Le Cloud computing : Infrastructure

Introduction

Le cloud computing est mode de partage de serveurs. Il est principalement constitué de 3 couches : l'application qui est directement à l’usage de l’utilisateur. La plateforme qui fait le lien entre la couche applicative applicative et matériel, elle est composé d’outils permettant la gestion et le contrôle des infrastructures matériel.
